Media Queries

Media queries allow us to make out sites responsive. We can use media queries to apple CSS styling only when cerian conditions apply. For instance, our sites can look different depending on the size of the screen, or viewport a user is viewing our content with. We can also change the way our site appears if a user is printing our website, or using a screen-reader!

Using media queries and responsive design allows us to ensure that our site looks and works no matter how the user is viewing it.

And, as more and more people use more and more devices to browse the internet, integrating media queries into website is becoming a commin, widespread practice!
